# PRD.md — Dodla D2C App: Product Requirements

## Problem Statement
Dodla Dairy is a large regional dairy brand (Hyderabad) with strong distribution but no direct digital relationship with end consumers. Provilac (Pune) has demonstrated that a premium D2C milk subscription app can create high LTV, low churn, and strong brand loyalty. Dodla needs an equivalent product.

## Target User
- Urban household in Hyderabad / Tier-1 AP/TS cities
- Orders dairy daily — currently from kirana or milk booth
- Smartphone-first, UPI-comfortable
- Values freshness, convenience, and predictability

## Core Value Proposition
> "Fresh Dodla dairy at your doorstep — subscribe once, never think about it again."

## Feature Requirements

### P0 — Must Have (MVP)
| Feature | Description |
|---|---|
| OTP Login | Phone number → mock OTP (555555) → logged in |
| Home Dashboard | Today's delivery status, wallet balance, quick actions |
| Product Catalog | Browse all Dodla SKUs, filter by category |
| Subscription Management | Add, remove, modify qty, pause individual subscriptions |
| Delivery Calendar | Monthly view; tap date to pause/resume; vacation mode |
| Wallet System | Balance display, recharge (simulated), auto-deduct, transaction history |
| Profile | Name, address, delivery instructions |

### P1 — Should Have
| Feature | Description |
|---|---|
| Loyalty Coins (Pro Coins) | Earn 1 coin/₹10; redeem for wallet credit; tier system |
| Referral Program | Shareable code; ₹50 for both parties |
| Low Balance Warning | Alert when wallet < 3 days of deliveries |
| Recipes Section | 6+ recipes using Dodla products with ingredients |
| Delivery Instructions | No-bell, leave at door, give to security, etc. |

### P2 — Nice to Have
| Feature | Description |
|---|---|
| Farm Stories | Transparency content about Dodla's farms |
| Order History | Full past order log with invoice simulation |
| Notifications | Simulated notification feed |
| Out of Stock UI | Badge + restock date |

## Success Metrics (post-launch)
- D30 retention > 70% (subscription model drives this)
- Wallet recharge frequency: 2x/month average
- NPS > 60
- Referral rate > 15% of new users
